🔥🏺 第02讲 史前技术与希腊思想

核心句

史前技术说明技术先于科学而存在；希腊思想说明科学的关键起点不是工具更复杂，而是用自然解释自然、追问本源和建立抽象模型。

史前技术
  • 旧石器和新石器：石器、火、墙、房屋、编织、衣服、矛、弓箭、陶器。
  • 可明确视为早期科学萌芽的领域：天文、计数数学、医学、博物学。
  • 博物学背景：采集者需要认识植物、动物、季节、水源、地形和材料。
  • 技术的社会条件：农业、水利、集体协作、政府、王国之间互相强化。

火的意义
  • 火是可控能源。
  • 功能：取暖、照明、驱兽、烹饪、扩展食谱、杀灭病菌和寄生虫。
  • 烹饪改变消化方式，减少消化能耗，可被理解为人类身体和文化协同演化的例子。
  • 火也是普罗米修斯神话中的文明象征：技术既是能力，也是风险。

哥贝克力石阵与农业次序
  • 传统想象：先定居 - 农业 - 村落繁荣 - 宗教中心。
  • 哥贝克力石阵提供反向可能：先有宗教或仪式中心 - 为支持建筑和仪式而需要稳定食物 - 促进定居和农业。
  • 复习重点：文明起源不是单线因果，宗教、组织、农业、建筑可能互为条件。

巨石阵
  • 英国南部索尔兹伯里巨石阵。
  • 需要长期观测太阳和月亮，掌握地平天文学。
  • 可以理解为巨大天象仪或时钟。
  • 说明早期天文知识服务于历法、仪式、农业和社会组织。

六个独立出现的文明
  1. 美索不达米亚，约公元前3500年后。
  2. 埃及，约公元前3400年后。
  3. 印度河流域，约公元前2500年后。
  4. 中国黄河流域，约公元前1800年后。
  5. 中美洲，约公元前500年左右。
  6. 南美洲，约公元前300年后。

希腊为何重要
  • 希腊是西方文明史和科学史的重要起点。
  • 类似中国先秦，短时间出现大量思想家，文学、艺术、哲学、数学、天文等集中爆发。
  • 后世科学的基本框架、目标和终极问题在希腊已有雏形。
  • 怀特海式说法：后世哲学可看作柏拉图的注脚。

文明起点的通则与希腊特色
  • 通则：命名对象、分类对象、建立对象关系。
  • 希腊特色：寻找本源、终极原因、单一因果、机械模型。

哲学两大门类
  • 自然哲学：世界本源、世界规律、人与自然关系。
  • 生存哲学：好的生活、有意义的生活、幸福如何可能。

前苏格拉底自然哲学
  • 泰勒斯：万物本源是水。
  • 阿那克西曼德：本源是"无定者"或 apeiron。
  • 阿那克西美尼：本源是气。
  • 恩培多克勒：土、气、火、水四根。
  • 毕达哥拉斯：数是世界秩序的关键，数学化倾向开始凸显。

可背论证链

史前技术不等于现代科学：人类很早有复杂工具和大型工程 - 这些依靠经验、观测、组织和手艺 - 不必预设现代科学理论 - 所以技术可以先于科学。

希腊成为科学起点：不是因为希腊最早有技术 - 而是它把世界当作可被理性解释的对象 - 用自然解释自然 - 追问本源和因果 - 建立抽象模型。
