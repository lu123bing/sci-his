🚂🏭 第09讲 工业革命与工业文明

核心句

工业革命使科学、技术、资本、能源和机器生产结合起来，科学从沉思性的解释活动转变为改造世界的生产力量，科学主义也由此获得社会基础。

工业革命关键词
  • 蒸汽机、煤、纺织机、钢铁、交通、工厂制度。
  • 1712年，纽科门蒸汽机。
  • 1765年，哈格里夫斯珍妮纺纱机。
  • 1781年，瓦特改良蒸汽机。
  • 1807年，富尔顿蒸汽轮船。
  • 1814年，斯蒂芬森蒸汽火车。
  • 1876年，贝尔电话。
  • 1885年，本茨汽车。

蒸汽机的意义
  • 蒸汽机把热能转化为机械功。
  • 蒸汽机需要煤和金属工业支持。
  • 蒸汽机 + 纺织机 = 工厂化生产和劳动纪律。
  • 纺织业扩张带来钢铁、运输、原料和市场需求。
  • 蒸汽机车使原料和产品运输加速，市场扩大。

工业文明
  • 工业文明不是单一技术，而是一整套社会结构：机器、工厂、能源、资本、市场、城市、劳动分工、交通网络。
  • 人和自然都被纳入效率、产量、成本、控制的逻辑。
  • 工业革命使“技术”从手艺和工具扩展为大规模系统。

科学在社会中的角色变化
  • 古代科学常类似哲学和艺术，偏形而上、沉思和解释。
  • 近代以后，科学与技术联姻，科学成为新技术的母体。
  • 科学的新功能：形而下地改造自然和社会。
  • 科学家成为世界的阐释者，科学技术成为世界的改造者。

科学主义
  • 科学主义建立的标志性事件之一：根据牛顿引力理论预言海王星存在。
  • 科学主义维度：
    • 科学 = 真理。
    • 科学 = 理性。
    • 科学 = 进步。
    • 科学 = 唯一正确解释方案。
  • 认知模式：机械自然观、机械论、还原论、决定论。
  • 风险：把科学方法误认为万能价值，把复杂社会和生命问题机械化处理。

中国的科学主义
  • 中国科学主义常在公众科学素养不足、现代化焦虑和救亡图存背景下展开。
  • 科学被寄托为国家富强、社会进步和破除迷信的力量。
  • 需要区分：尊重科学不等于把科学当作新宗教。

📝 课堂补充：话语权的转移——从教会到科学

科学革命不仅是科学进步，更是话语权的根本转移：关于"这个世界是什么样子"的解释权，从教会转移到了科学家手中。伽利略强调，除了阅读《圣经》（上帝拯救人类的书），人类还必须阅读"自然之书"（上帝创造世界留下的物理法则），因为《圣经》只能告诉人们如何上天堂，却不能告诉人们"天体如何运行"——这为早期的自然哲学家（科学家）争取到了合法性与话语权。1992年教皇保罗二世正式为伽利略平反：伽利略当年说话时教会拥有绝对话语权；360年后教皇道歉时，教会在自然规律的解释上已完全失去话语权。

📝 课堂补充：医疗话语权的争夺——中医为何被西医取代？

并非因为"原理更科学"：人们通常以为西医取代中医是因为细胞学说、细菌学等科学原理更正确、解释力更强。但老师反驳了这种绝对的技术决定论，指出西医的胜利本质上是社会建构与自上而下的话语权赋予。

1926年的社会认知差异：回到100年前，中国普通老百姓根本听不懂"细胞""细菌"，但对中医的"上火""受寒""阴阳失调"一听就懂，喝姜汤是普遍生活经验。因此，在当时的民间认知基础（缺省配置）下，西医绝不可能仅靠"原理更科学"就自然战胜中医。

梁启超的"医疗事故"：梁启超笃信西医和科学，在协和医院做手术时遭遇严重医疗事故（医生误切了他健康的右肾导致病情恶化并最终离世），但他为了不伤害人民群众对刚刚兴起的西医的信念，坚决要求不把这个负面消息公之于众。

蔡元培"废除读经"与教育体制的西化：西医真正获得话语权，是因为民国时期政府高层和知识分子普遍接受了西方科学思想。教育总长蔡元培推行"废除读经"，中国学生不再系统学习传统文化，而是学习西方的数理化生。政府自上而下推行了废除中医运动，赋予西医绝对话语权。如今医学院基本全是西医系统（中医只是选修），而中医药大学里西医却是必修课。

📝 课堂补充：工业革命对社会生态的彻底重构

核心方法：用自己的经验去展开逻辑推理——不要背诵现成结论，而要一步步推导蒸汽机带来的连锁反应。

蒸汽机 + 珍妮纺纱机 → 翻天覆地的社会变化：①空间重构（工厂诞生）——过去纺纱是单人手动的家庭作坊；蒸汽机能同时带动上百台机器，必须专门修建巨大厂房，催生了现代"工厂"。②人机关系的主客体反转（异化）——家庭作坊里人是机器的主人，累了随时休息；工厂里蒸汽机一旦开动就停不下来，人变成机器的仆人，必须死死盯在岗位上。③时间的空间化与定量化——工厂催生了"上下班"制度和"工人"阶级，仅仅靠教堂敲钟已不够，个人家庭开始需要钟表，时间被高度数学化和定量化，深刻约束了现代人的生活。④劳动的简单化——社会学家郑也夫指出：农民的劳动是"复杂劳动"（需懂二十四节气、播种收割全周期）；现代工人的劳动是"简单劳动"（如卓别林《摩登时代》中工人只负责拧一个螺丝）。极度劳动分工让技能培训变得极简，从而能大规模、快速地再生产工人。

亚当·斯密与全球化扩张：亚当·斯密在《国富论》中强调"劳动分工"（英国人擅长织布，德国人擅长种土豆，交换把蛋糕做大）。机器大生产导致布匹产量暴增、成本锐减，英国本地市场饱和后必然寻求外销。廉价洋布大量倾销到中国江南地区，直接击垮了中国传统家庭手工纺织业，进而逼迫中国诞生了试图与之对抗的早期民族资本家。

可背论证链

工业革命改变科学角色：蒸汽机等技术重组生产 - 工厂和资本扩大技术需求 - 科学与技术结合产生新工业 - 科学家获得社会权威 - 科学主义把科学从方法提升为现代社会的最高解释权威。
