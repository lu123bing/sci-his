3_03：为什么学历史、人文社科要求与价值观

2. 有些人在写作中习惯使用“我们”来代替“我”，这一现象被老师用来说明对学生的什么要求？

原答案： 作为一个个体去思考

考点扩展：

  • 属于人文社科课程的基本要求。
  • “我们”有时会遮蔽个人判断，容易变成集体立场或权威话语。
  • 老师强调学生要能提出自己的判断，而不是只复述教材、老师或社会共识。
  • 与复习笔记中的“作为个体思考”“学会论证一件事，而不是宣称一件事”对应。

可转化简答句：

人文社科训练要求学生作为独立个体思考，用理由和史料支持判断，而不是只以“我们认为”代替个人论证。

3. 你喜欢吃辣的，我喜欢吃甜的，我们没有高低贵贱之分，这被老师用来阐述什么概念？

原答案： 文化相对主义

考点扩展：

  • 文化相对主义强调：不同文化、习惯、价值偏好不应简单用单一标准区分高低贵贱。
  • 它不是说所有价值判断都取消，而是提醒我们先理解不同文化自身的背景和逻辑。
  • 在科学与文明史中，这一概念有助于避免“现代西方科学唯一优越”的简单叙事。
  • 可联系李约瑟问题：中国古代知识体系不能简单用“落后/先进”评价，而要分析其制度、功能和目标。

易错点：

  • 文化相对主义 ≠ 没有任何判断标准。
  • 它强调理解差异，反对粗暴等级化。

4. 上课讲到化学家哈伯和克拉拉的故事，是为了说明学历史可以：

原答案： 重塑价值观

考点扩展：

  • 哈伯既是合成氨的重要科学家，也参与化学武器研究。
  • 克拉拉反对科学服务大规模杀戮，常被视为科学伦理问题的象征人物。
  • 这个故事说明：历史不是只提供事实，也会改变我们对“进步”“成功”“科学家责任”的理解。
  • 可联系课程主线：科学技术具有双重性，不能只以效率、产量、诺奖评价科学。

可转化简答句：

学历史可以重塑价值观，因为历史展示同一科学知识如何在不同制度和伦理选择中产生拯救或毁灭的后果。

5. 田老师课上提到了哪两个欧洲女科学家？

原答案： 克拉拉、居里夫人

考点扩展：

  • 克拉拉·伊梅瓦尔：与哈伯故事相关，重点在科学伦理、化学武器、科学家的社会责任。
  • 居里夫人：现代科学史中重要女性科学家，可联系放射性研究、科学共同体和女性科学家处境。
  • 本题可能考课堂记忆，但可扩展为“科学史不只是男性科学家英雄史”。
