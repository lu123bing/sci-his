最后一次：科学主义、化学、农业、环境伦理

1. 田松老师在讨论科学主义时指出，被视为科学主义两大重要“堡垒”的是哪些领域？

原答案： 医学、现代农业（尤其是现代农业技术）。

考点扩展：

  • 医学和现代农业常被认为最能体现科学技术“拯救生命、提高产量”的正面形象。
  • 但课程强调：即便是医学和农业，也不能只从技术有效性评价。
  • 医学涉及伦理、个体经验、制度分配。
  • 现代农业涉及化肥、农药、种子资本、水资源、土壤生态和社会分化。
  • 这正是批判科学主义的重点：尊重科学不等于把科学当作唯一价值权威。

2. 《寂静的春天》出版后，美国是否立即全面禁止所有化学农药？

原答案： A（错误说法）。
正确史实： 美国并未立即全面禁止所有化学农药；1972年才禁止DDT在农业中的使用。
补充：

  • B 正确：推动了美国环保运动和EPA建立；
  • C 正确：蕾切尔·卡森 获得过重要学术荣誉；
  • D 正确：农药公司曾试图阻止相关内容传播。

考点扩展：

  • 《寂静的春天》出版于 1962 年。
  • 作者蕾切尔·卡森是海洋生物学家、博物学家。
  • 该书揭示 DDT 等农药对生态系统和人类健康的风险，推动现代环保运动。
  • 注意表述严谨：它推动环保运动，但没有使美国“立即全面禁止所有化学农药”。

3. 法国大革命期间被处死、令拉格朗日感叹“这样的头脑一百年也长不出来”的科学家是谁？

原答案： 安托万-洛朗·拉瓦锡。

考点扩展：

  • 拉瓦锡是近代化学奠基者。
  • 关键词：氧气、燃烧、定量实验、反燃素说、质量守恒、化学命名体系。
  • 法国大革命中被处死，体现科学与政治制度的复杂关系。
  • 复习时注意人物匹配：拉瓦锡不是哈伯，不是李比希。

4. 寂静的春天 的重要性主要体现在哪里？

原答案：

  • 揭露化学农药生态危害；
  • 推动现代环保运动；
  • 促使社会反思“科技万能论”和科学主义。

考点扩展：

  • 《寂静的春天》的核心不是简单反科学，而是反思技术后果。
  • 它强调生态系统中的食物链、反馈、累积和长期影响。
  • 它消解了科学技术的“神性”：科学技术不总是进步和拯救，也可能制造环境风险。
  • 可与机械自然观对比：机械自然观看短期控制，生态学视角看系统后果。

可背简答：

《寂静的春天》通过揭示农药在生态系统中的累积和扩散，推动现代环保运动，也促使人们反思科技万能论和科学主义。

5. “下列哪一项是现代化学奠基者拉瓦锡的核心贡献？”

原答案： C
即：通过定量实验研究氧气与燃烧，推翻燃素说。

考点扩展：

  • 燃素说认为燃烧是物体释放燃素。
  • 拉瓦锡通过定量实验说明燃烧与氧气有关，推动化学从定性经验转向定量实验科学。
  • 拉瓦锡的意义不仅是一个发现，而是建立近代化学语言、命名和守恒观念。
  • 易考：拉瓦锡对应“近代化学”，李比希对应“农业化学/肥料”，哈伯对应“合成氨”。

6. “在生态学视角里，《寂静的春天》重点体现了哪一生态原理？”

原答案： B
即：毒素沿食物链累积富集（生物富集）。

考点扩展：

  • 生物富集：有毒物质通过食物链逐级累积，高营养级生物体内浓度更高。
  • 这说明不能只看短期剂量，还要看长期、系统和跨物种影响。
  • 与生态自然观相关：自然是关系网络，不是孤立对象的集合。
  • 期末可能考判断：农药“有效杀虫”不等于“生态安全”。

7. 关于 弗里茨·哈伯 的表述，哪些正确？

原答案： A、B、D

  • A：人工固氮深刻改变农业与社会；
  • B：体现科学技术的“双刃剑”特性；
  • D：克拉拉·伊梅瓦尔 常被视为科学伦理问题象征人物之一。 错误项：
  • C 错，因为哈伯确实参与了毒气武器研究。

考点扩展：

  • 哈伯-博世法使工业化合成氨成为可能。
  • 合成氨的双重用途：
    • 化肥：提高粮食产量，支撑人口增长和工业化；
    • 炸药/军工：服务战争。
  • 哈伯还参与一战化学武器研究。
  • 克拉拉反对化学武器并自杀，体现科学伦理冲突。

可背简答：

哈伯是科学技术悖论的典型：同一合成氨技术既能生产化肥、养活人口，也能服务炸药和战争，说明科学评价不能脱离制度、战争、资本和伦理。

8. 关于 克拉拉·伊梅瓦尔 ，科学史最关注的是哪件事？

原答案： A
即：她反对化学武器，并于一战期间自杀。

考点扩展：

  • 克拉拉常被视为科学伦理问题的象征人物。
  • 她的故事用于提醒：科学家的责任不只是追求技术成功，也包括反思技术用途。
  • 可与“学历史可以重塑价值观”联系。
  • 易错：不要只把她记成“哈伯的妻子”，重点是科学伦理与化学武器。

9. “中国在2005年曾出现一场关于环境伦理的重要讨论，其讨论主题是什么？”

原答案： D
即：征服自然。

考点扩展：

  • “征服自然”体现典型现代性和机械自然观语言。
  • 环境伦理会反思：人类是否应把自然仅仅当作资源和控制对象。
  • 可联系：
    • 非人类中心主义；
    • 都江堰“顺应自然”；
    • 农药化肥导致生态风险；
    • 生态自然观强调共存、反馈、长期后果。

10. 关于 尤斯图斯·冯·李比希 ，哪项表述正确？

原答案： C
即：他曾建议染料厂加入含铁盐，使蓝色染料更加明显。

考点扩展：

  • 李比希关键词：有机化学之父、农业化学之父、肥料工业之父。
  • 他强调植物生长需要碳、氢、氧、氮以及钾、磷等矿物元素。
  • 他提出应向土壤补偿植物带走的营养，奠定农业化学和肥料学基础。
  • 李比希体现科学与工业、农业实践之间的结合。
