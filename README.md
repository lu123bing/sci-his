# 科学与文明史 — 复习知识卡片 Web 版

基于 Markdown 笔记自动生成的交互式知识点复习网站，支持卡片式浏览、全文搜索、快捷键操作。

## 主要功能

- **智能瀑布流卡片布局**：知识点以卡片形式呈现。采用了弹性计算布局（Flex + JS 算法），根据每个卡片的文本字数和当前行内的平均字数动态计算并分配宽度（提供 0.5、0.75、1、1.25、1.5 比例档位）。大幅度拉平了同一行内部卡片的高度落差，视觉上更加整齐平稳。
- **可折叠目录栏**：左侧章节导航，支持折叠以扩大阅读区域
- **粘性搜索栏**：顶部毛玻璃搜索框，滚动时始终可见，输入时自动展开
- **高级全文聚合搜索 & 高亮**：
  - **基于小节与卡片的结果聚合**：搜索结果不再零散分布，而是以“小节大标题”为基础进行卡片层级的聚合预览，并剔除未能命中的无关卡片。
  - **联合关键词 (AND) 搜索**：支持输入多个关键词（空格隔开），精准匹配必须同时具有所有检索维度的核心卡片。
  - **按相关性算分排序**：记录关键词在导言、标题以及每一个卡片文本中的命中次数并累加计算 Score，根据热度排序，搜索结果更为精准。
  - **双重智能高亮机制**：点击搜索结果实现无缝跳转。在详情页，只有真正“包含全部检索词”的合格卡片才会被强效高亮并自动展开折叠；仅“碰巧包含个别词”的不合格卡片只会用半透明底色弱化标识，避免视线感染。
- **SPA 无刷新导航**：点击目录或搜索结果时，使用 PJAX 技术无缝切换页面，不中断阅读体验
- **卡片折叠控制**：一键展开/折叠全部卡片
- **响应式适配**：适配不同屏幕尺寸，折叠目录后自动增加一列卡片

## 快速开始

```bash
# 安装依赖（仅构建时需要）
npm install

# 构建静态网页
node build.js

# 本地预览
npx serve docs
```

构建后所有文件位于 `docs/` 目录，可直接部署到任意静态托管服务（GitHub Pages、Vercel、Netlify 等）。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `f` | 聚焦搜索框，并滚动到页面顶部 |
| `h` | 切换左侧目录栏的折叠/展开 |
| `↑` / `↓` | 在搜索结果中上下选择（需先聚焦搜索框） |
| `Enter` | 跳转到当前选中的搜索结果 |
| `Esc` | 关闭搜索下拉菜单 |
| `Ctrl` + `←` / `→` | 切换到上一篇/下一篇笔记 |

```
│   └── assets/

```

## 技术栈

- **构建**：Node.js 脚本，解析 Markdown 生成静态 HTML 和 JSON 结构化索引
- **前端**：原生 HTML/CSS/JavaScript，零框架依赖
- **样式**：Flex 动态高度适配布局、CSS Variables 主题变量、毛玻璃效果
- **交互**：Fetch API + History API 实现 SPA 导航、TreeWalker 与多重正则节点遍历实现智能高亮
- **搜索**：预生成 JSON 索引，支持按需多关键词联合提取、局部权重评分、DOM 节点动态分配过滤
