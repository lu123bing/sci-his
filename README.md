# 科学与文明史 — 复习知识卡片 Web 版

基于 Markdown 笔记自动生成的交互式知识点复习网站，支持卡片式浏览、全文搜索、快捷键操作。

## 主要功能

- **瀑布流卡片布局**：知识点以卡片形式呈现，自动适配屏幕宽度分栏
- **可折叠目录栏**：左侧章节导航，支持折叠以扩大阅读区域
- **粘性搜索栏**：顶部毛玻璃搜索框，滚动时始终可见，输入时自动展开
- **全文搜索 & 高亮**：支持关键词搜索，搜索结果自动截取前后 50 字上下文，点击跳转并高亮匹配内容
- **SPA 无刷新导航**：点击目录或搜索结果时，使用 PJAX 技术无缝切换页面，不中断阅读体验
- **卡片折叠控制**：一键展开/折叠全部卡片
- **响应式适配**：适配不同屏幕尺寸，折叠目录后自动增加一列卡片

## 快速开始

```bash
# 安装依赖（仅构建时需要）
npm install

# 构建静态网页
node build.js

# 本地预览
npx serve docs
```

构建后所有文件位于 `docs/` 目录，可直接部署到任意静态托管服务（GitHub Pages、Vercel、Netlify 等）。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `f` | 聚焦搜索框，并滚动到页面顶部 |
| `h` | 切换左侧目录栏的折叠/展开 |
| `↑` / `↓` | 在搜索结果中上下选择（需先聚焦搜索框） |
| `Enter` | 跳转到当前选中的搜索结果 |
| `Esc` | 关闭搜索下拉菜单 |
| `Ctrl` + `←` / `→` | 切换到上一篇/下一篇笔记 |

## 项目结构

```
.
├── build.js              # 构建脚本（Markdown → HTML）
├── update_ui.js          # UI 升级辅助脚本
├── src/
│   ├── template.html     # 页面模板
│   └── assets/
│       ├── style.css     # 样式表
│       └── main.js       # 前端交互逻辑（搜索、导航、快捷键）
├── docs/                 # 构建输出目录（可直接部署）
├── *.md                  # 原始笔记文件
└── README.md
```

## 技术栈

- **构建**：Node.js 脚本，解析 Markdown 生成静态 HTML
- **前端**：原生 HTML/CSS/JavaScript，零框架依赖
- **样式**：CSS Grid 瀑布流布局、CSS Variables 主题变量、毛玻璃效果
- **交互**：Fetch API + History API 实现 SPA 导航、TreeWalker 实现文本高亮
- **搜索**：预生成 JSON 索引，客户端实时搜索
